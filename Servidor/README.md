# API Restful da aplicação "Bazar do Livro"

<h2> Execução </h2>

Para executar o servidor é necessário a instalação do typescript:

`npm i -g typescript`

Do nodemon:

`npm i -g nodemon`

E das dependências gerais:

`npm i`

Após isso pode-se executar o arquivo `main.ts` om o nodemon:

`nodemon main.ts`
